About IMIK

Ibaybay mo, Isasalin ko, is a web application that allows transliteration between Latin script and Baybayin, an ancient Philippine script.

Our Mission

Our goal with this project is to develop a web application
that transliterates between Latin script and Baybayin script. We want
to apply concepts from automata theory that we learned in class, specifically
finite state automata. We hope to increase appreciation and understanding
of the ancient Baybayin script by making it accessible for people to
read and write this script from our past.
